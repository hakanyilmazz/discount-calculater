# Spring Rest API Example - Discount Calculater

## Requirements
---------------------------------------
1- Maven (3.8.5+)

2- Java JDK (17)

3- Postman

---------------------------------------

## How to build?
---------------------------------------
1- Set JAVA_HOME and MVN_HOME to Environment variables of your operating system.

2- Run this script on terminal
```
$mvn clean install
```

## How to run?
1- Run this script on terminal
```
$mvn spring-boot:run
```
